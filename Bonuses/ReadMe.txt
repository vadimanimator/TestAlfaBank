	Программа формирования отчета по бонусам для сотрудников

Программа формирует отчет по бонусам для сотрудников за указаный период.

При запуске из консоли следует передать 3 параметра:
1) Формат вывода ("console" или "file") или ("консоль" или "файл") без кавычек
2) Начальная дата расчета
3) Конечная дата расчета
Даты вводятся в виде ДД.ММ.ГГГГ
	Пример запуска программы:
Bonuses.exe console 01.04.2015 07.11.2017
где Bonuses.exe - имя исполняемого файла

При некорректном вводе параметров или при запуске исполняемого файла напрямую, программа запросит необходимые данные.

Если выбран формат вывода "file", то результат сохранится в папке c программой с именем "Начальная дата-Конечная дата.csv", например:
01.04.2015-07.11.2017.csv

Для корректной работы программы необходимы файлы:
	Сотрудники.csv
 - таблица сотрудников со следующими данными:
Код;ФИО;Дата приема на работу;Дата увольнения;Код бонуса;Оклад
При наличии нескольких кодов бонуса, они перечисляются через запятую
	Договора.csv
 - таблица заключенных договоров со следующими данными:
Код;Код сотрудника;Дата Договора;Сумма договора

Даты также вводятся в виде ДД.ММ.ГГГГ

Эти файлы должны находиться в одной папке с программой
При обработке файлов подразумевается, что заголовки присутствуют (Важно)

Если не русские символы не отображаются, в настройках консоли выберете шрифт "Luciada Console"

P.S. Разделителем в файлах должна быть точка с запятой (;)
